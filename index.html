C) Installation Guide - top

For End Users (MSIX Package):

  1. Download the MSIX installer package from your purchase confirmation email or from CodeCanyon
  2. Double-click the .msix file to begin installation
  3. If prompted, click "Install" to proceed
  4. Windows will automatically install all required dependencies including .NET 8.0 Runtime
  5. Once installed, launch "Mail Extractor Pro" from the Start Menu
  6. On first launch, you'll see the License Activation window
  7. Enter your Envato purchase code to activate (see License Activation section)
Easy Installation: The MSIX package includes everything you need. No manual .NET runtime installation required!

For Developers (Source Code):

  1. Extract the source code ZIP file to your desired location
  2. Install Visual Studio 2022 (17.0 or later) with:
    • .NET desktop development workload
    • Windows App SDK development workload
  3. Open Mail Extractor.sln in Visual Studio
  4. Restore NuGet packages (Visual Studio will do this automatically)
  5. Configure your license settings in LicenseConfig.cs (see Configuration section)
  6. Build and run the project (F5)
Developer Note: Make sure to configure LicenseConfig.cs with your Envato API credentials before distributing your build.

D) License Activation - top

Mail Extractor Pro uses Envato's official API for secure license verification. Your purchase code is verified directly with Envato to ensure authenticity.

How to Activate Your License:

  1. Launch Mail Extractor Pro
  2. On first startup, the License Activation window will appear automatically
  3. Enter your Envato purchase code in the text field
  4. Click "Activate License" button
  5. Wait for verification (requires internet connection)
  6. Once activated, you'll see a success message with your buyer information
  7. Click "Continue" to start using the application

Finding Your Purchase Code:

To find your Envato purchase code:

  1. Log into your Envato Market account
  2. Go to your Downloads page
  3. Find "Mail Extractor Pro" in your purchases
  4. Click "Download" and select "License certificate & purchase code"
  5. Your purchase code will be shown on the certificate (format: xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx)
Helpful Links in Activation Window:
  • Purchase on Envato: Don't have a license? Click to buy one
  • Watch on YouTube: Video tutorial on activation and usage
  • Find Purchase Code: Direct link to Envato's help guide
  • Contact Support: Email support if you need help

License Security Features:

Mail Extractor Pro implements advanced security to protect against piracy:

  • API Verification: All licenses verified through Envato's official API
  • Hash Verification: SHA-256 hash prevents tampering with stored license data
  • Machine Binding: License is bound to your computer to prevent sharing
  • Periodic Re-verification: License is re-checked every 7 days online
  • Integrity Checks: Automatic detection of license data tampering
Important Notes:
  • Each purchase code can only be used on one computer at a time
  • Internet connection required for activation and periodic re-verification
  • If you change computers, you may need to deactivate on the old machine first
  • For multiple computers, you need separate licenses

E) Configuration for Developers - top

If you've purchased the source code version, you can customize the license system for your own Envato product.

Step 1: Get Your Envato API Token

  1. Visit https://build.envato.com/api/
  2. Click "Create a Token"
  3. Give it a name (e.g., "Mail Extractor License Verification")
  4. Select the following permissions:
    • View and search Envato sites
    • Verify purchases of your items
  5. Copy the generated token

Step 2: Configure LicenseConfig.cs

Open Mail Extractor\LicenseConfig.cs and update the following values:

public static class LicenseConfig
{
    // Set to true to enable license verification, false for free version
    public static readonly bool RequireLicense = true;

    // Your Envato API Personal Token
    public static readonly string EnvatoApiToken = "YOUR_ENVATO_API_TOKEN_HERE";

    // Your Envato Item ID (find it in your item URL)
    public static readonly string EnvatoItemId = "YOUR_ITEM_ID_HERE";

    // Your product name
    public static readonly string ProductName = "Mail Extractor Pro";

    // Your support email
    public static readonly string SupportEmail = "support@yourcompany.com";

    // Your YouTube tutorial link
    public static readonly string YouTubeLink = "https://www.youtube.com/watch?v=YOUR_VIDEO_ID";

    // Your Envato product page
    public static readonly string EnvatoProductLink = "https://codecanyon.net/item/your-product/YOUR_ITEM_ID";
}
Finding Your Item ID:
Your item ID is in the URL of your CodeCanyon product page:
https://codecanyon.net/item/product-name/12345678
In this example, 12345678 is your Item ID.

Security Configuration

The license system includes these configurable security settings in LicenseService.cs:

  • VERIFICATION_INTERVAL_DAYS: How often to re-verify license online (default: 7 days)
  • Hash generation includes machine name for machine-binding
  • All verification is done through Envato's official API
Security Best Practices:
  • Never commit your API token to public repositories
  • Use environment variables or secure configuration for production
  • Test license activation thoroughly before releasing
  • Consider code obfuscation for additional protection

H) Frequently Asked Questions - top

Q: Do I need an internet connection to use the app?
A: Yes, internet is required for initial license activation, periodic re-verification (every 7 days), and email/DNS verification features. The core extraction functionality works offline once activated.

Q: Can I use my license on multiple computers?
A: Each license is bound to one computer. For multiple computers, you need separate licenses or an extended license.

Q: What if I change my computer?
A: Contact support to transfer your license to a new machine. You may need to deactivate on the old machine first.

Q: How do I update the application?
A: Updates will be available through CodeCanyon. Download the new version and install over the existing installation. Your license will remain active.

Q: What file formats can I load?
A: Currently supports .txt text files. Additional formats may be added in future updates.

Q: Is my data secure?
A: Yes. All data is processed locally on your computer. No data is sent to external servers except for email/DNS verification features (which only send the email address being verified).

Q: Can I customize the application?
A: If you purchased the source code version, yes! The application is built with .NET/WinUI 3 and fully customizable.

I) Troubleshooting - top

License Activation Issues:

Problem: "Could not verify license" error
Solution:

  • Check your internet connection
  • Verify you entered the correct purchase code (no extra spaces)
  • Ensure the purchase code is for Mail Extractor Pro (not another product)
  • Contact support if the issue persists

Problem: "License already in use" error
Solution:

  • Each license can only be active on one computer at a time
  • If you have multiple computers, you need separate licenses
  • Contact support to transfer your license

Performance Issues:

Problem: Slow performance with large files
Solution:

  • Ensure your system meets the recommended requirements (8GB RAM)
  • Close other applications to free up memory
  • Consider processing large files in smaller batches

Application Won't Launch:

Solution:

  • Verify Windows 10/11 is up to date
  • Ensure .NET 8.0 Runtime is installed
  • Try reinstalling the application
  • Check Windows Event Viewer for error details
